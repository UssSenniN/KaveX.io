💻 Portfólio: Projeto m2V

Simulador de orçamento para materiais de piso

📌 Sobre o projeto

O m2V é um simulador web que ajuda a calcular o custo estimado de instalação de pisos, levando em conta preços por metro, argamassa e rejunte. Os dados permanecem na tela até que o usuário decida limpar.

⚙️ Funcionalidades

  • Entrada de valor por metro quadrado
  • Exibição automática de estimativas de custo
  • Armazenamento dos dados com localStorage
  • Botão para limpar os dados manualmente

🧪 Tecnologias utilizadas

  • HTML, CSS e JavaScript
  • GitHub Pages para hospedagem
  • localStorage para persistência de dados
