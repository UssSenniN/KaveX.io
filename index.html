💻 Portfólio: Projeto m2A.

Calculadora de Cerâmica

📌 Sobre o projeto

O m2A é um simulador web que ajuda a calcular o custo estimado de instalação de pisos, levando em conta preços por metro, argamassa e rejunte.

📌 Objetivo do projeto

O m2A Apresentar uma ferramenta digital que agiliza o cálculo de materiais para instalação de cerâmica e porcelanato, garantindo orçamentos precisos e eficientes para vendedores e empresas de materiais de construção.

⚙️ Funcionalidades

  • ✅ Agilidade no atendimento: Cálculo instantâneo, reduzindo o tempo de orçamento.
  • ✅ Precisão nos cálculos: Evita erros que podem gerar insatisfação e retrabalho.
  • ✅ Maior conversão de vendas: Clientes confiam mais em orçamentos bem estruturados.
  • ✅ Parcelamento inteligente: Sugestão automática de opções de pagamento conforme o valor total.
  • 📈 Aumento de produtividade: Vendedores podem atender mais clientes em menos tempo.

🧪 Tecnologias utilizadas

  • HTML, CSS e JavaScript
  • GitHub Pages para hospedagem
