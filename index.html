💻 Portfólio: Projeto m2V

Simulador de orçamento para piso, argamassa e rejunte

📌 Sobre o projeto

O m2V É uma calculadora simples para estimar os custos de materiais na aplicação de pisos, incluindo argamassa e rejunte. O usuário informa os dados, visualiza os resultados e os mantém salvos até que deseje limpar.

⚙️ Funcionalidades

  • Cálculo estimado baseado em área e valor por metro
  • Exclusão manual dos dados pelo usuário
  • Interface direta e responsiva

🧪 Tecnologias utilizadas

  • HTML, CSS e JavaScript puro
  • GitHub Pages para publicação
